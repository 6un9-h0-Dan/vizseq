Installation

#Requirements

VizSeq requires Python 3.6+.

#Installation

You can install VizSeq via one-line pip command:

$ pip install vizseq

Or install it from source:

$ git clone https://github.com/facebookresearch/vizseq
$ cd vizseq
$ pip install -e .
